# little_endian

C and Python developer from Kyiv building Windows tools, low-level utilities, and embedded side projects.

I like practical software that talks directly to the system: Windows audio routing, parsers and CLI tools in C, VR/media tooling, and small ESP8266 and STM32 projects.

## Current focus

- Building [winappaudiorouter](https://github.com/rostok2112/winappaudiorouter), a Python library for per-app Windows audio routing
- Expanding Windows-first media and automation tooling around VRChat and streaming
- Learning more DevOps and deployment workflows

## Featured projects

- [winappaudiorouter](https://github.com/rostok2112/winappaudiorouter)  
  Python library for routing Windows app input and output devices through the Windows audio policy APIs.

- [vrc-media-gateway](https://github.com/rostok2112/vrc-media-gateway)  
  Toolkit for streaming YouTube, SoundCloud, Telegram, Spotify, and images into VRChat with FastAPI, nginx, browser extensions, and HLS.

- [libcalc](https://github.com/rostok2112/libcalc)  
  C library for tokenizing mathematical expressions, converting them to RPN, and evaluating them with a stack calculator.

- [PowerWatch](https://github.com/rostok2112/PowerWatch)  
  ESP8266-based power restoration notifier with RGB indicators, buzzers, and Telegram alerts.
