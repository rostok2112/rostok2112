# little_endian

Python/Django backend developer with a Master's degree in Computer Engineering.

I work on backend systems professionally, and outside work I spend most of my time on the same things I actually enjoy building: VRChat and VR tools, Windows-focused utilities, low-level experiments, and embedded projects.

## Focus

- Backend development with Python and Django
- VRChat, VR, and Windows-first tooling
- Embedded systems and microcontroller projects
- Practical libraries, automation, and developer utilities

## Featured work

- [winappaudiorouter](https://github.com/rostok2112/winappaudiorouter)  
  Python library for routing Windows app input and output devices through the Windows audio policy APIs.

- [vrc-media-gateway](https://github.com/rostok2112/vrc-media-gateway)  
  Windows-first toolkit for streaming YouTube, SoundCloud, Telegram, Spotify, and images into VRChat with FastAPI, nginx, browser extensions, websocket RPC, and HLS.

- [libcalc](https://github.com/rostok2112/libcalc)  
  C library for tokenizing mathematical expressions, converting them to RPN, and evaluating them with a stack calculator.

- [PowerWatch](https://github.com/rostok2112/PowerWatch)  
  ESP8266-based power-restoration notifier with RGB indicators, buzzers, and Telegram alerts.

- [diploma](https://github.com/rostok2112/diploma) + [diploma-server](https://github.com/rostok2112/diploma-server)  
  A SteamVR full-body-tracking project split into a low-level module and a Python server for moving data into VR workflows.
